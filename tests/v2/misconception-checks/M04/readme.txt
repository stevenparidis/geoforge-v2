M04 — Thrust not steep dip (POSITIVE)

Tests that when a thrust fault has a steep dip (70°) with a validation_note set,
clicking the event feat-item reveals the .validation-note-pill.

Misconception: Confusing thrust faults (low-angle, <45°) with reverse faults (steeper dip).
A 70° dip on a 'thrust' is geologically implausible and should be flagged.
