M07 — Throw/displacement conflation (POSITIVE)

Tests that when a fault has throw equal to displacement on a non-vertical fault (validation_note set),
clicking the event feat-item reveals the .validation-note-pill.

Misconception: Throw (vertical component of slip) equals net displacement. This is only true
for vertical faults (dip = 90°). At any other dip angle, net displacement > throw.
At 45° dip: net displacement = throw / sin(45°) ≈ 7.1m, not 5m.
