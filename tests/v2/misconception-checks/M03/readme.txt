M03 — Strike/dip independence (POSITIVE)

Tests that when a fault event has a validation_note set for violating the right-hand rule
(strike and dip_direction are inconsistent), clicking the event feat-item reveals .validation-note-pill.

Misconception: Strike and dip direction are treated as independent measurements, ignoring the
right-hand rule constraint: dip_direction = (strike + 90) mod 360.
