M06 — Intrusion older than host (POSITIVE)

Tests that when a dyke intrusion has a validation_note set (intrusion claimed to be older than host),
clicking the intrusion feat-item reveals the .validation-note-pill.

Misconception: An intrusion can predate the rocks it cuts. In reality, by the principle of
cross-cutting relationships (Hutton / relative dating), an intrusion must be younger than
the rocks it intrudes.
