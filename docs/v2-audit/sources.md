## How to Use These Sources During the Audit

### Workflow
1. For each GeoForge feature subtype, open the Primary source(s) listed above.
2. Load the corresponding GeoForge v1 screenshot from `docs/v2-audit/v1-screenshots/<class>/`.
3. Compare GeoForge output against the reference diagram on these specific axes:
   - **Geometry**: correct shape, proportions, dip angles, relative positions of bodies
   - **Labelling**: correct geological terminology where labels are shown
   - **Implied geology**: does the rendered scene imply the correct geological history and process?
   - **Distinguishability**: can a student tell this subtype from adjacent subtypes (e.g. sill vs laccolith; disconformity vs angular unconformity)?

### Priority guidance
- **Primary** sources are the audit standard — GeoForge output is measured against these.
- **Secondary** sources provide supplementary geometry detail and should be consulted when a Primary source is ambiguous or lacks a specific subtype diagram.

### Sidecar files
Each `docs/v2-audit/references/<class>/*.txt` file contains:
- The exact URL
- Access date
- A description of available diagrams
- A note on what specifically to check in GeoForge against that source
