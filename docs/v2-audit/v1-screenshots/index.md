# v1 Screenshots Index

This directory is populated during Phase A.2 (v1 screenshot capture).

Each subdirectory corresponds to a GeoForge feature class:

- `faults/` — normal, reverse, thrust, strike-slip, oblique, listric
- `folds/` — anticline, syncline, monocline
- `layers/` — horizontal, dipping, multi-layer
- `intrusions/` — dyke, sill, batholith, laccolith
- `unconformities/` — angular, disconformity, nonconformity
- `mineralisation/` — porphyry, orogenic gold, VMS, skarn, epithermal

Screenshots should be named: `<subtype>-<view>.png` (e.g. `normal-fault-cross-section.png`).
